# React Frontend

## Development

1. Install dependencies:
```bash
npm install
```

2. Start development server:
```bash
npm run dev
```

The dev server will run on http://localhost:3000 and proxy API requests to the Flask backend on port 5001.

## Production Build

1. Build the React app:
```bash
npm run build
```

2. The built files will be in `dist/` and will be served by the Flask backend.
